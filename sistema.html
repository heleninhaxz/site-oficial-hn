Sistema Operacional

Conceito: Um sistema operacional é um software que gerencia recursos de hardware e proporciona uma interface para interação entre usuários e programas.

Sistemas Operacionais - Funções Básicas

  • Gerenciamento de dispositivos periféricos (teclado, vídeo, impressora, mouse, ...);

  • Gerenciamento de arquivos;

  • Gerenciamento de memória;

  • Gerenciamento de Processos;

  • Interpretação de comandos do usuário;

  • Administração de serviços.

  • Segurança do computador.

Principais Tópicos

  • Intermediário entre hardware e software: Age como uma camada intermediária que permite que o software interaja com o hardware do sistema de forma eficiente.

  • Coordenação e gerenciamento de recursos: Controla recursos como processadores, memória, dispositivos de armazenamento e dispositivos de entrada/saída para garantir que sejam usados de forma eficaz pelos aplicativos.

  • Interface para interação com o usuário e programas: Fornece uma interface, que pode ser uma linha de comando ou uma interface gráfica, para que os usuários e os programas possam interagir com o sistema.

  • Permite execução de aplicativos e processos: Gerencia a execução de programas e processos, permitindo que múltiplos aplicativos sejam executados simultaneamente.

  • Controla o acesso e alocação de memória: Gerencia a alocação e desalocação de memória para os processos em execução e otimiza o uso da memória disponível.

  • Gerencia o armazenamento e acesso a arquivos: Organiza e controla o armazenamento e acesso a arquivos no sistema de armazenamento, garantindo integridade e segurança dos dados.

  • Facilita a comunicação com dispositivos de entrada/saída: Gerencia a comunicação entre o sistema e dispositivos de entrada/saída, como teclado, mouse, impressoras, etc.

  • Oferece uma plataforma para desenvolvimento de software:Fornece um conjunto de APIs e ferramentas para que os desenvolvedores possam criar e executar aplicativos no sistema operacional.

  • Responsável pela inicialização do sistema: Gerencia o processo de inicialização do computador, carregando o sistema operacional na memória e preparando-o para uso.

  • Fornece serviços de rede e comunicação: Oferece suporte para conexão de rede e comunicação, permitindo que dispositivos se comuniquem entre si através de redes locais ou da internet.

  • Proporciona suporte para múltiplos usuários: Permite que múltiplos usuários possam interagir com o sistema simultaneamente, cada um com suas próprias configurações e preferências.

  • Pode ser encontrado em diferentes tipos de dispositivos: Desde computadores pessoais até dispositivos móveis, servidores e sistemas embarcados, o sistema operacional é uma parte essencial de uma ampla gama de dispositivos digitais.

  • Evolui continuamente para acompanhar avanços tecnológicos e demandas dos usuários: Está em constante desenvolvimento para incluir novos recursos, melhorias de desempenho, correções de segurança e suporte a novas tecnologias, garantindo sua relevância e eficácia ao longo do tempo.

Sistemas Operacionais - Exemplos

  • Windows: É o sistema mais usado em computadores pelo mundo, o Windows foi lançado originalmente na década de 1980 como uma evolução do antigo MS-DOS.

  • Linux: É um sistema de código aberto, que é bastante conhecido por ser altamente personalizado pelos usuários, que assim conseguem adaptá-lo para o uso desejado.

  • Android: Lançado em 2008, é um sistema operacional para celulares e tablets desenvolvido pelo Google.

  • iOS: é o sistema operacional móvel da Apple, lançado em 2007, sendo utilizado no iPhone e no iPod Touch, além de fazer integração com outros sistemas da marca, como o macOS, do Mac, e o tvOS, da Apple TV.
